# my-agents

## Setup ##

## 1ère étape : ##

# placer le dossier agents dans :

Pour Linux :
```sh
~/.config/opencode/
```
Pour MacOS :
```sh
~/Library/Application Support/opencode/
```
Pour Windows :
```sh
C:\Users\<USER>\.config\opencode\
```

## 2nd étape ## 

Remplacer le fichier opencode.jsonc du dossier précédent avec celui du repo
